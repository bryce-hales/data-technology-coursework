## Dashboard Summary

The rental analysis dashboard includes visuals for:

- Average rent by number of bedrooms
- Average rent by number of bathrooms
- Rental price versus square feet
- Percent of listings with more than two bedrooms
- Average rental price by state
- Rental price filtering
- State filtering
- Dataset subcategory buttons for apartments, homes, and short-term rentals

## Skills Practiced

- Applying a branded dashboard theme
- Using slicers and filters
- Comparing rental prices across states
- Creating scatter plots for rental price versus square footage
- Segmenting listings by bedroom and bathroom count
- Building a dashboard layout with cards, charts, slicers, and brand assets

## Related Power Query and DAX Concepts

The project included data preparation steps such as setting field types, replacing blank values, calculating price per square foot, and converting Unix timestamps to datetime values. It also used DAX measures for listing count, average rent, median rent, average square footage, and average price per square foot.
