# Coursework Datasets

This folder documents datasets used across coursework, notebooks, spreadsheets, SQL labs, and Power BI reports.

## Dataset Inventory

| Dataset | Format | Used For |
| --- | --- | --- |
| `iris.csv` | CSV | Statistics, scatter plot, and classification-style practice. |
| `seattle_weather.csv` | CSV | Power BI statistical charts using precipitation, rain status, and temperature data. |
| `superstore.csv` | CSV | Power BI sales analysis by category, sub-category, customer, product, date, and quantity. |
| `transactions.csv` | CSV | Transaction-level practice dataset. |

## Dataset Summaries

### seattle_weather.csv

- Rows: 25,551
- Columns: `DATE`, `PRCP`, `TMAX`, `TMIN`, `RAIN`
- Used to analyze rainfall, precipitation bins, yearly precipitation totals, and temperature relationships.

### superstore.csv

- Rows: 9,994
- Columns include order information, customer fields, product category fields, sales, quantity, discount, and profit.
- Used to analyze sales by category, sub-category, product, customer, date, and order quantity.

### transactions.csv

- Rows: 60
- Columns: `TransactionID`, `Date`, `EmployeeID`, `Bought`, `PaymentMethod`, `RecurringCustomer`
- Used for transaction and customer behavior practice.
